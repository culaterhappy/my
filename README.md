    过了一会儿，隔扇缓缓拉开，奈津子走了进来。有刚洗完澡的淡淡的香味。
    奈津子关了灯，钻进了旁边的被窝。
    两个人都沉默着，寂静持续了一段时间。
    这时，奈津子小声说:“喂。”她叫我。
    “嗯?”
    “牵手吧。”奈津子说。我转头看向奈津子，奈津子闭着眼睛望着天花板。
    我默默地从被子里伸出手，奈津子也同样从被子里伸出手握住。
    “以前，我们经常这样牵着手睡觉呢。”
    能感受到奈津子手的温暖。
    “刚才……”


    我把刚才在厨房和有希交谈的事告诉了奈津子。
    “真是的，有希这一点是很敏锐的。“”
    “我也是这么想的。“”
    “‘女人’变强了的感觉，啊···。“说了这句话，奈津子一下子笑了出来。“但是，我果然变了吧。和那个人在一起。“”
    奈津子非常自然地说要和那个人在一起，这让我感到轻微的震惊，但什么也没说。
    之后我们沉默了一会儿，奈津子又说：
    “喂。“
    “嗯？“”
    “现在是什么心情？“”
    “好像和人妻睡在一起的房间里，是不是？“”
    奈津子按照我的感觉这样回答，
    “啊哈，的确，也许是这样。“”
    说着咯咯地笑了。
    “你想做？“”
    “那是当然的。“”
    “不行，忍耐一下。“奈津子转向我，用恶作剧的口气说。“不过，我可以帮你释放一下，只要你不做什么奇怪的事。”
    我抬起一点被子，奈津子从旁边的被子里溜了进来，从后面变成了被我抱住的姿势。奈津子的头发散发出洗发水的香味。
    “以前是这样睡的吧。“
    “是啊.。“



    现在，两个人都穿着睡衣，而以前经常在一张床上，两个人光着身子这样贴着睡。
    奈津子睡衣的领子下面露出了一条金项链。看到这一幕，我明白了现在抱着的这个身体是田山的，抱着奈津子的力气更大了。
    通过睡衣传来奈津子的体温和触感，我不由自主地勃起了。
    “呵呵。”奈津子意味深长地笑了。“感觉有什么不对的。”
    “没办法啊，谁让我这样抱着‘人妻’的。”
    我把鼻子贴在奈津子的头发上，在她耳边低语。
    我想从睡衣上揉奈津子的胸，奈津子按住我的手说:“不行，我说了吧，乖一点。”说。
    “好吧，就这些。”
    “是啊，你不会随便去碰别人的妻子吧?”
    奈津子边说边拍了拍我的手。
    即使奈津子这么说，在以前也只是开玩笑，但现在很现实。
    “不过，你这么想要我，我很高兴。”
    “奈津子也越来越坏了。”
    我有点不满地说。
    “啊，是吗。”奈津子又扑哧一声笑了。“那个人也说过，这样做才是对你的爱，所以，让你更想要我。”
    “……。”
    我沉默了一会儿，抱着奈津子。
    “奈津子。”
    “什么?”
    “能拜托你一个事情吗?”
    “什么?”
    “这次能抱紧我吗?”
    “好啊。”
    奈津子说完，转向我，把我的头抱在胸前。穿着睡衣在脸上感觉到奈津子丰满的胸围的触感。奈津子的身上散发着芳香的肥皂香味。
    “这样行吗?”奈津子问我。
    “啊。”
    我把脸埋在奈津子的怀里，感到脑袋发晕般的幸福感，久久不动。
    “奈津子……。”



    “什么?”
    “那个……我就这样自己做可以吗?”
    我这么问奈津子，奈津子沉默了一下，说:“好啊。”回答道。
    我把睡衣的裤子和内裤一起褪下一点，拿起枕边的纸巾，再次把脸埋在奈津子的胸前，用右手握住自己的阴茎，慢慢地移动起来。
    奈津子静静地抚摸着我的头。
    “你一边做一边在想什么呢?”奈津子问我。“今天早上我和那个人做爱的样子?
    “嗯！！！”我一边动手一边回答。
    “兴奋吗?”奈津子在我头上问道。
    “嗯，特别兴奋。”
    我把鼻子贴在奈津子的胸前，说道。项链碰到额头了。
    “总是那样被那个人抱着，我···。”
    我默默地继续右手套弄着。
    “心情好起来了吗?”
    奈津子问我。
    “嗯。”
    我这么一说，奈津子又温柔地抚摸着我的头。
    “你想要我吗?”
    “想要，非常想要。”
    我在奈津子的怀里回答。
    “我很高兴能被你追求。今后你也会一直这样为我着迷吗?”
    我被奈津子抱着头，点点头。
    “想要更多……爱着那个人的我吗？”
    “想要……想要奈津子哦。”
    我一边动手，一边把嘴唇贴在奈津子的胸前，呻吟着说。
    “无论我怎样成为那个人的女人，他也会爱我吗?”
    “嗯！！！是的是的！！！”


    我加快了右手的动作。
    “我的身体一直在寻找那个人，不是吗？”
    奈津子紧紧抱着我的头，在我耳边低语。
    “现在还想要吗?”我问奈津子。
    “嗯，我现在也想要那个人，下面已经湿透了。”
    “真的……。”
    “是啊，真的。一想到那个人，就想要。高兴吗?”
    我再次点头。
    “可怜的老公……再也不能抱我了。”
    奈津子再次在我耳边低语。
    我想起了今天早上看到的奈津子的样子。
    皮肤白皙通透的腿被高高举起，田山的阴茎在其根部骄傲地从私处反复进出。
    再也不能进入那个里面了…。一想到自己的阴茎再也不会被奈津子体内的那种温暖所包围，疯狂的嫉妒就变成兴奋的风暴在脑海中奔涌，我剧烈地射精。
    我被奈津子抱着，上气不接下气地呼吸着，奈津子一直温柔地抚摸着我的头发。
